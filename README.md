# Movie Search App

A movie search application built with HTML, CSS, and JavaScript.

This app allows users to search for movies and view information such as posters, release year, ratings, and plot details using a movie database API.

## Preview

![Demo](assets/demo.gif)

## Features

* Search movies by title
* Display movie posters
* Show release year
* Display movie ratings
* View movie details
* Responsive design

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Movie API (OMDb API)

## How to Use

1. Enter a movie title in the search box.
2. Click the search button.
3. Browse the search results.
4. View movie information and details.

## Project Status

Completed as a practice project for improving API integration, asynchronous JavaScript, and DOM manipulation skills.
